Перед использованием API Диадока через COM необходимо зарегистрировать типы данной сборки в системе.

Зарегистрировать сборку можно при помощи командного файла Diadoc_Reg.bat 
Отменить регистрацию сборки можно при помощи командного файла Diadoc_Unreg.bat 

В случае, если Windows блокирует регистрацию сборки по причине ее загрузки 
из Интернета, следует использовать командный файл Diadoc_Unblock.bat для 
восстановления работоспособности библиотеки.
